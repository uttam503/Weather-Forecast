# 🌦️ SkyCast Pro – Weather Forecasting Web Application

## 📌 About the Project

SkyCast Pro is a modern and responsive weather forecasting web application developed using **HTML, CSS, and JavaScript**. The application uses **WeatherAPI** to provide accurate real-time weather information and a 7-day forecast for cities around the world. Users can search for any city or use their current location to instantly view weather details through a clean, intuitive, and responsive interface.

---

## ✨ Key Features

* 🔍 **City Search** – Search weather information for any city worldwide.
* 📍 **Current Location Support** – Automatically fetch weather data using the device's GPS location.
* 🌡️ **Real-Time Weather** – Display current temperature, weather condition, and weather icon.
* 📅 **7-Day Forecast** – View the weather forecast for the next seven days.
* 💧 **Weather Details** – Shows humidity, wind speed, visibility, and atmospheric pressure.
* 📈 **Temperature Chart** – Interactive hourly temperature graph using Chart.js.
* 📱 **Responsive Design** – Optimized for desktop, tablet, and mobile devices.
* 🎨 **Modern Glassmorphism UI** – Attractive interface with smooth animations and clean design.
* ⚡ **Fast API Integration** – Fetches live weather data using asynchronous JavaScript (Fetch API).
* ❌ **Error Handling** – Displays appropriate messages for invalid city names or network issues.
