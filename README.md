# Weather-Forecast
🌦️ SkyCast Pro - Weather Forecasting Website

SkyCast Pro is a modern and responsive weather forecasting website that provides real-time weather information for any city worldwide. Users can search for a city or use their current location to view accurate weather data, including temperature, humidity, wind speed, pressure, visibility, a 7-day forecast, and an interactive temperature chart.

✨ Features
🔍 Search weather by city name
📍 Get weather using current location
🌡️ Real-time temperature and weather condition
💧 Humidity, Wind Speed, Pressure & Visibility
📅 7-Day Weather Forecast
📈 Interactive Temperature Chart (Chart.js)
📱 Fully Responsive Design
🎨 Modern Glassmorphism UI
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
WeatherAPI
Chart.js
Font Awesome
📸 Preview

A clean and user-friendly interface with real-time weather updates and beautiful weather cards.

🚀 How to Run
Clone the repository.
Add your WeatherAPI key in config.js.
Open index.html in your browser.
Search for any city or use the location button to view weather information.
📌 API Used
WeatherAPI (Real-time Weather & Forecast)
